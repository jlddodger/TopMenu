Welcome to TopMenu by Aperture Logic™.

TopMenu is a functionality clone of PowerMenu, but designed to be compatible with Windows 7 and 64-bit systems. TopMenu is under early development. It may crash or cause other problems (though I currently use it everyday with only rare issues).

Technical preview version:

  • TopMenu32.msi
  • TopMenu64.msi

Note for x64 users: Install both files to the same folder
